## AHC045

### 概要

- 各都市の存在範囲が入力で与えられる
- M 個のサイズが与えられるので、都市をグループ分けして、そのグループ間では連結になるように道路を敷設する
- 占いを Q 回行うことができ、都市集合 C (2 <= |C| <= L) を与えて、最小全域木の辺を入手可能

### 制約

### 解法

1. N * (N - 1) / 2 本の辺の長さの大小を占いで求めて、辺の長さの範囲を絞る
2. 中心をその頂点とみなし、x + y が小さい順にグループ分け
3. 2 ~ 3 グループ破壊して再構築する焼きなましを行う
    - その時、最初の始点を最も端の方にある頂点として作成すると良さそう
